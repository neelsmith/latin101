# Latin word order is *not* random!

A frequent pattern:

- subject-object-verb (SOV) **canis hominem mordet**.

Other possibilities:

- object-subject-verb (OSV) **hominem canis mordet**.
- verb-object-subject (VOS) **mordet hominem canis**.
