# Noun cases: review


- case recognized from *ending*
- cases indicate *function*
    - *nominative* : the "naming case"
    - *genitive* : the "connecting case"

### Example

*IVLIA AVGVSTA* **TITI AVGVSTI** *F(ILIA)*

> *Nominative*: names Julia Augusta

> **Genitive**: connects Titus to Julia
